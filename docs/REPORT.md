## Описание выполненных работ
1. **Развёртывание CMS Hugo:** Установлена расширенная версия Hugo v0.159.0 на ОС Windows.
2. **Настройка темы:** Использована тема Ananke, настроены параметры `baseURL` для корректного отображения ресурсов на GitHub Pages.
3. **Наполнение контентом:** Создан раздел с материалом «Введение», в который перенесён текст выпускной квалификационной работы.

   * [cite_start]Отражена цель исследования: создание электронного курса на платформе Moodle[cite: 8].
   * [cite_start]Включены данные эмпирического исследования 2025 года об уровнях толерантности среди студентов[cite: 5, 25].
